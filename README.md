# Open Source Audit Project

## Introduction
This project focuses on understanding the concept of Open Source Software using Python and Linux. It combines both theoretical knowledge and practical implementation through shell scripting.

The main objective is to explore how open source systems work and how basic tasks can be automated in a Linux environment.

---

## Scripts Overview

1. System Identity Report  
   Displays system details like user name, kernel version, uptime, and date.

2. File Analyzer  
   Checks whether a file exists and shows its line count and size.

3. Word Count Tool  
   Takes a sentence as input and counts the number of words.

4. Simple Calculator  
   Performs basic arithmetic operations like addition, subtraction, multiplication, and division.

5. File Backup Script  
   Creates a backup copy of a given file.
